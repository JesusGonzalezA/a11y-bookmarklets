A11y Bookmarklets

Accessibility audits for humans & AI agents

👁 Visual overlays for humans 🤖 Structured JSON for AI agents

How to install

  1. Drag any bookmarklet button below to your bookmarks bar
  2. Navigate to the page you want to audit
  3. Click the bookmarklet — visual overlays appear instantly
  4. Open the browser console to see the JSON data (window.__a11y)

Bookmarklets

For AI Agents

AI agents can use these bookmarklets via the MCP server or by injecting the scripts directly with Playwright.

MCP Server Setup

{
  "mcpServers": {
    "a11y-bookmarklets": {
      "command": "npx",
      "args": ["@bookmarklets-a11y/mcp-server"]
    }
  }
}

Agent Skill

npx skills add weaaare/bookmarklets-a11y@a11y-bookmarklets

Direct Playwright Usage

// Via browser_evaluate in Playwright MCP:
(() => {
  return window.__a11y.headings.audit();
})()
